Monolith Mode

Sophisticated path-tracing that identifies color clusters and merges them into singular, optimized vector paths. This mode is essential for keeping file sizes small and shapes clean for production.

Lego Mode

Every single pixel is translated into a discrete SVG rectangle. While heavy on performance, it provides a perfect 1:1 mapping for technical grid-based designs or voxel-like extrusions.

The Architecture

GLORP is powered by a custom Contour Meshing Engine. Using NumPy-accelerated processing, the engine performs a high-speed boundary scan to detect pixel transitions. A Greedy Search algorithm then traces these boundaries to build resolution-independent paths with minimal vertex counts. It ensures crisp edges and mathematical precision without using neural networks.
