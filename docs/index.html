    The Architecture

    GLORP utilizes a custom Contour Meshing Engine written in Python and compiled to WASM. It performs boundary analysis to minimize vertex counts while maintaining pixel grid integrity.

    FAQ

    Everything runs locally in your browser using Pyodide (WebAssembly). No images are ever uploaded to a server.
    Monolith creates connected paths which results in much smaller SVG file sizes. Lego is only for specific grid-based editing.
    The web version handles up to 50 files comfortably. For unlimited batch processing, use our Desktop version on Itch.io.
    GLORP fully supports alpha channels. Transparent pixels are ignored, and semi-transparent ones retain their opacity values.
    Harry Tsang is the lead engineer behind the high-performance contour algorithms that make GLORP 3.0.0 possible.
