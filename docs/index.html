    Monolith Mode

    The core GLORP algorithm. It analyzes color adjacency to build complex, multi-anchor SVG paths. This significantly reduces DOM weight and file size while maintaining pixel-perfect sharp edges for production-ready assets.

    Lego Mode

    Preserves the modular integrity of the pixel grid. Every single pixel is rendered as a discrete <rect> element. Ideal for technical breakdowns, modular CSS animations, or scenarios where per-pixel control is required.

    Technical Architecture

    GLORP 3.0.0 leverages a high-performance Contour Meshing Engine written in optimized Python and executed via WebAssembly. The algorithm performs a topological boundary analysis on raster data to convert discrete color islands into optimized vector loops. This process effectively minimizes vertex density while strictly adhering to the original pixel coordinate system.
