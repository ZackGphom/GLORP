GLORP: THE RULES

1:1 Pixel Ratio. Upload the file exactly as you saved it from your editor (Aseprite, Photoshop, etc.). Do not resize it.

PNG Only. JPEG compression creates invisible color noise. Noise = broken SVG.

No Post-FX. No blurs, soft shadows, or gradients. Only solid colors.

The golden rule: if the image looks blurry or too big for its resolution, GLORP will see it as trash data and crash. Keep it original.

    What is GLORP?

    GLORP is an online tool for converting pixel art to SVG. It is made for people who need clean vectors from pixel-based artwork without leaving the browser.

    Processing happens locally in your browser. Your images are not sent to a server, which keeps the workflow fast and private.

    It is useful for artists, game asset creators, NFT and collectible makers, poster and print work, and developers who want a quick pixel to vector pipeline.

    Extra details

    Local by design

    Images are decoded and converted in the browser. There is no upload step and no backend conversion queue.

    Batch upload stays

    You can drop many files at once. The queue still runs file by file so the export flow stays predictable.

    Smart warnings

    Large scenes trigger a warning before conversion, so you can stop before the browser gets overloaded.

    Output modes

    Monolith makes denser SVG paths, Lego keeps the pixel grid exact, and WebP exports a raster version when SVG is not needed.
